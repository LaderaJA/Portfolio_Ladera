Project Description

AHome is a home service platform developed using the Django framework. It connects homeowners with skilled workers for various household services, including repairs, maintenance, and renovations. The web app features job postings, user authentication, and a job application system. A real-time chat and location-based job search function are also integrated. This project showcases my expertise in backend development, database management, and user experience design.
